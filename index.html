Política de Privacidade

Aplicativo Docent

Última atualização: 02/12/2025

Esta Política de Privacidade descreve como o aplicativo Docent ("nós", "nosso", "aplicativo") coleta, usa e protege as informações dos usuários. Ao utilizar o aplicativo, você concorda com os termos descritos nesta política.

1. Informações que Coletamos

1.1 Informações fornecidas pelo usuário

Podemos coletar informações inseridas diretamente por você, incluindo, mas não se limitando a:

  • Nome completo
  • Endereço de e-mail
  • Dados de cadastro necessários para o funcionamento do aplicativo
  • Informações inseridas voluntariamente durante o uso

1.2 Informações coletadas automaticamente

O aplicativo pode coletar automaticamente dados como:

  • Modelo do dispositivo e sistema operacional
  • Endereço IP
  • Idioma do dispositivo
  • Interações e dados de acesso dentro do aplicativo

1.3 Serviços de terceiros

O Docent pode utilizar serviços externos, como Firebase ou APIs de terceiros. Esses serviços podem coletar dados conforme suas próprias políticas de privacidade.

2. Como Utilizamos as Informações

As informações coletadas podem ser utilizadas para:

  • Operar e aprimorar o aplicativo
  • Personalizar a experiência do usuário
  • Armazenar dados e preferências
  • Garantir segurança e prevenção de fraudes
  • Atender solicitações e suporte ao usuário

3. Compartilhamento de Informações

Não compartilhamos seus dados com terceiros para fins comerciais. As informações podem ser compartilhadas apenas quando:

  • For necessário para o funcionamento do aplicativo
  • Houver exigência legal
  • Houver consentimento explícito do usuário

4. Armazenamento e Segurança

Seus dados são armazenados com medidas apropriadas de segurança. Caso utilizemos o Firebase ou serviços externos, os dados podem ser processados em servidores internacionais.

5. Permissões Utilizadas

O aplicativo pode solicitar permissões, como:

  • Acesso à internet
  • Notificações
  • Outras permissões específicas necessárias ao funcionamento

6. Direitos do Usuário

Você pode solicitar:

  • Acesso, correção ou exclusão de seus dados
  • Encerramento da conta
  • Revogação de permissões concedidas

Para isso, envie uma solicitação para: wendel.alvessousa02@gmail.com

7. Retenção dos Dados

Manteremos seus dados apenas pelo tempo necessário para o funcionamento do aplicativo ou conforme exigido por lei.

8. Alterações nesta Política

Podemos atualizar esta Política de Privacidade periodicamente. Mudanças significativas serão informadas dentro do aplicativo.
